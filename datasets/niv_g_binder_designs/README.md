# NiV-G Binder Design Dataset Layout

This directory contains templates and scripts for transparent reporting of the computational NiV-G binder design campaign.

- `target/` - target and epitope metadata templates
- `raw/` - raw model outputs (user-supplied, not tracked in templates)
- `processed/` - cleaned and filtered metrics tables
- `final_submission/` - final candidate metadata templates
- `figures/` - generated summary plots
- `scripts/` - lightweight pandas/matplotlib workflow scripts
